# Multi-Docs Conversational Chatbot

Upload multiple DOCX files and have natural conversations with your documentation.

## Features
- Multi-file upload
- Multi-session chat
- Conversational memory
- Natural responses (OpenAI API)
- Evidence-based answers

## Requirements
- Python 3.10+
- OpenAI API Key
- Install: `pip install -r requirements.txt`

## Run Locally
```bash
streamlit run app.py
